## Setup
1. Configure `.env` in the `backend/` directory.
2. Run `npm install` in both `backend/` and `frontend/` directories.
3. Seed the admin account using `node seed.js` in the `backend/` directory.
4. Start the servers using `npm run dev`.

Refer to [PROJECT_DOCS.md](PROJECT_DOCS.md) for full details on environment keys, installation, and seeding.
